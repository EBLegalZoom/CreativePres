03

What is HTML?

  • HyperText Markup Language
  • Uses the DOM or Document Object Model to make up the skeleton and structure of every webpage
  • Within the DOM, uses tags to define different parts of content i.e. this is "a heading" or "a paragraph," etc
  • We then use CSS to style those tags

07

What is CSS?

  • Cascading Style Sheets
  • The paint, wallpaper, and interior design of the web
  • Controls how HTML elements look
  • Handles colors, fonts, sizes, spacing, and layout
  • Brings websites to life with animations and effects
08

Use CSS Classes to Style Tags

They start with a . in CSS and use class="" in HTML!

The CSS:

.yellow { color: yellow; }
.pink { color: pink; }
.white { color: white; }

The HTML:

<h1 class="yellow">This h1 is yellow!</h1>
<p class="pink">This p is pink!</p>
<h3 class="white">This h3 is white!</h3>
